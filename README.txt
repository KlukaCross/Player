Программа Sorting_PLAYER
Плеер с особой функцией.
Суть проекта заключается в том, чтобы сортировать музыку на компьютере для удобства по жанрам, настроению, тематике, и пр.

Основная часть.
В левую верхнию строку записывается диск или папка, содержащие музыкальные файлы. После нажатия 'Искать на диске',
программа сканирует и выдаёт слева список всех папок, имеющих музыкальные типы файлов.
При нажатии на одну из папок справа отображается список аудио. Если выбрать один из треков, то он начинает
воспроизводиться. В верхней строке подписанны все кнопки:
Пуск - останавливает воспроизведение и начинает его заново
Стоп - останавливает воспроизведение 
Пауза - ставит на паузу
Воспроизведение - продолжение
Предыдущая - начинает воспроизведение предыдущего файла (если файл стоит самый первый в списке, то
программа воспроизодит самый последний файл в списке)
Следующая - начинает воспроизведение следующего файла (если файл стоит самый последний в списке, то
программа воспроизодит самый первый файл в списке)
Повтор - имеет 3 стадии: 'отключен', 'повтор 1 раз', 'играть бесконечно'. По умолчанию стоит 'повтор отключен'. При одном нажатии устанавливается функция 'повтор 1 раз',
т.е. после завершения трек повторяется 1 раз. При двойном нажатии устанавливается функция 'играть бесконечно' - трек играет бесконечное количество раз.
Но если совершить действие во время проигрывания, функция распространиться только на следующий трек.
Описание функций горячих клавиш в верхней строке:
Горячие клавиши дублируют функции кнопок, расположенных над ними.(работают только при английской раскладке)

Особенности плеера.
В нижнем виджете отображенны 3 вкладки, подразумевающие папки, в которых будут сортироваться музыкальные файлы по жанрам, настроению и др. параметрам.
Для начала надо ввести путь к существующей папке (сохранить) и присвоить ей горячую клавишу, т.к. у нас 3 папки, мы можем присвоить 3 горячие клавишы из предлагаемого списка,
появляющиемся во вспывающем окне.
А дальше мы слушаем музыку и сортируем музыкальные файлы по папкам с помощью горячих клавиш.
Таким образом мы теперь легко можем найти музыку, отвечающую нашему настроению или другим потребностям.


p.s. К сожалению, в данном проекте не удалось добиться одной из немаловажных функции плеера - последовательное воспроизведение треков.
